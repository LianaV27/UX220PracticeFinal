Tim Berners Lee

The inventor of the world wide web also invented...

HTML

HTML: Hypertext Markup Language gives us the standard structure for web pages. It is often used with other languages such as CSS: Cascading Style Sheets and JavaScript.

URL

URL: A Uniform Resource Locator, aka a Web Address, carries a reference to its location and a method to retrieve it.

HTTP

HTTP: Hypertext Transfer Protocol is an application layer protocol that serves as the foundation for data communication on the web. For example, consider a hypertext document with hyperlinks.
